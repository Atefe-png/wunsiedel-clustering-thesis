# Interactive Dashboard

The interactive web map is maintained in a separate repository 
and deployed via GitHub Pages.

**Live map:** https://atefe-png.github.io/wunsiedel-dashboard/

**Source code:** https://github.com/Atefe-png/wunsiedel-dashboard

The dashboard displays all 2,673 POIs with cluster assignments 
from K-Means, DBSCAN, and HDBSCAN using Leaflet.js and 
CartoDB Positron tiles.
